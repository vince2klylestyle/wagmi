================================================================================
CRITICAL FINDINGS
================================================================================

1. NO STARTUP VALIDATION
   - Bot starts without checking critical environment variables
   - Missing API keys cause silent failures (trading continues)
   - No pre-flight checks before live trading
   Risk: Accidental paper trades, missing alerts, invalid credentials
   Fix: 1 hour - Add validate_config() function

2. ENVIRONMENT MODE CAN BE BYPASSED
   - Paper vs. Live mode set by ENVIRONMENT variable
   - Can be overridden after bot starts
   - No immutable "lock" on mode selection
   Risk: Paper trading config doesn't match actual mode
   Fix: 2 hours - Add mode validation and locking

3. API KEYS NOT SANITIZED IN LOGS
   - Anthropic API key loaded without masking
   - Hyperliquid private key could leak in exceptions
   - Exception tracebacks logged without redaction
   Risk: API key leaks in log files or error reports
   Fix: 1 hour - Add SensitiveRedactor logging filter

4. NO CREDENTIAL FORMAT VALIDATION
   - Hyperliquid credentials accepted without checking format
   - Invalid addresses/keys won't fail until submission
   - No test of credentials before trading
   Risk: Invalid config discovered at worst time
   Fix: 1 hour - Add credential validation at startup

5. PROFILE OVERRIDES SILENT
   - Paper vs. Live profiles silently override env vars
   - User sets MAX_LEVERAGE=50 but live profile uses 25
   - No warning about which settings were changed
   Risk: Configuration doesn't match user expectations
   Fix: 30 min - Log all profile overrides with warnings

================================================================================
HIGH-RISK CONFIGURATION ITEMS
================================================================================

ENVIRONMENT=paper (default)
  - Default is safe but not validated
  - Can be overridden: export ENVIRONMENT=production

RISK_PER_TRADE=0.005 (0.5% in paper, 0.02 = 2% in live by .env.production)
  - Paper: Very aggressive (0.5%)
  - Live: 4x higher (2%) — dangerous for $400 account
  
MAX_LEVERAGE=25.0 (global cap)
  - Paper: 25x allowed
  - Live: .env.production uses 10x for $400 account (safer)
  - Risk tiers: high-risk assets capped at 12x

CIRCUIT_BREAKER_DAILY_LOSS_PCT=0.05 (5% of equity)
  - $10k account: stops at $500 loss
  - $400 account: stops at $20 loss
  - Tight, but appropriate for small accounts

MAX_CONSECUTIVE_LOSSES=5 (paper), 4 (live)
  - After 4-5 losses, circuit breaker trips for 2 hours
  - Prevents revenge trading

ENSEMBLE_CONFIDENCE_FLOOR=55.0
  - Very low floor (trades at 55%+ confidence)
  - Other gates (R:R, EV, fees) filter quality

LLM_MODE=0 (OFF by default)
  - Can be 0-5 for increasing autonomy
  - 0 = OFF, 2 = VETO_ONLY, 5 = FULL AUTO
  - Default safe (no LLM trading)
